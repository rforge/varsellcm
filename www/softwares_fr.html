Logiciels (packages R)

  • Le package R CoModes est un outil qui estime, sur des données qualitatives, un modèle de mélange de distributions par modes. Il s'agit de distributions multinomiales où les quelques paramètres libres correspondent aux croisements de modalités de plus grande probabilité. Ce package R est disponible sur R-forge ici.

  • Le package R Clustericat effectue une classification non supervisée des données qualitatives en utilisant l'extension par bloc du modèle des classes latentes et contient des fonctions utiles pour résumer les résultats et les principales dépendances conditionnelles ainsi que pour visualiser les paramètres. Il est disponible sur R-forge ici.
